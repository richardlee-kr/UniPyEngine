# UniPyEngine
UniPyEngine is unity-like 2D game engine based on pygame.

# Features
- AnimationClip
- Animator
- AudioClip
- AudioSource
- Collision
- GameObject
- Scene
- SceneManagement
- Sprite
- SpriteRenderer
- Transform
- UI
- Vector

# In Development

    - OnTriggerEnter

    - OnTriggerExit
